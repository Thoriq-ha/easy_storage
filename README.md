# Storage App 
Easy aplication for parking vehicles only with qr code and fingerprint scanner


## Installation

#### Step 1
Download or clone this repo by using the link below:

```bash
https://github.com/Thoriq-ha/easy_storage.git
```

#### Step 2
Download or clone this repo by using the link below:

```bash
flutter pub get 
```


#### Step 3
Download or clone this repo by using the link below:

```bashx
flutter pub run 
```
